PayID+CLI+termible == Developer ♥️

Initialize a new PayID:

payid init alice$xpring.money

Add a Crypto Address

crypto-address add XRPL MAINNET rw2ciyaNshpHe7bCHo4bRWq6pqqynnWKQg 722714325

Generate an Identity Key

keys generate identity-key

Sign the Address

payid sign

Inspect signatures on the loaded PayID:

payid inspect

Learn more about the CLI:

help
help payid
